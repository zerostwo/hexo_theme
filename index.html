hexo插件使用总结

post @ 2018-12-05

hexo插件简介

Hexo 有强大的插件系统，使您能轻松扩展功能而不用修改核心模块的源码。在 Hexo 中有两种形式的插件：

脚本（Scripts）

如果您的代码很简单，建议您编写脚本，您只需要把 JavaScript 文件放到 scripts 文件夹，在启动时就会自动载入。

插件（Packages）

如果您的代码较复杂，或是您想要发布到 NPM 上，建议您编写插件。首先，在 node_modules 文件夹中建立文件夹，文件夹名称开头必须为 hexo-，如此一来 Hexo 才会在启动时载入否则 Hexo 将会忽略它。

CSS

post @ 2018-12-01

CSS（Cascading Style Sheets即层叠堆积样式）是在HTML元素提供网页结构的地方，控制在该网页上呈现这些HTML元素的方式。样式表可以应用于任何网页，使编辑和管理多页网站的外观和布局变得更加容易。

CSS3是CSS的最新版本，包含许多强大的功能和属性，使Web设计人员可以更好地控制网页，而不必依赖于使用图像或外部JavaScript库。功能包括灵活的框，动画，多列，圆角，阴影，过渡，渐变，动画等等。

在这里，我将介绍内容几乎涵盖了CSS和CSS3的所有领域，包括flexbox，动画，图像过滤器，响应工具包，框架，库等等。
