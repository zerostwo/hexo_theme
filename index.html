基本概念

化学反应速率

IUPAC 物理化学部动力学分委员会推荐，以单位体积内反应进度随时间的变化率为反应速率

对于体积不变的反应，上式可以写为：

对化学反应，反应速率

<\p>
从上我们不难看出，反应速率的SI单位为。

### 基元反应

由反应物一步变化直接得到生成物的反应成为基元反应。
反应体系内物质浓度与反应速率的关系称为速率方程，一般通过实验来测定拟合。对于基元反应的速率方程为，这个关系就是质量作用定律（law of mass action）。质量作用定律只能直接作用于基元反应！

### 反应级数

由于速率方程是表示为反应物浓度某方次的乘积，即，式中各浓度项指数的加和称为反应级数（）。当时，称为级反应。
反应级数同样是通过实验测定拟合得到的。例如：蔗糖的水解反应，若不通过实验会得到其是个二级反应，可是实际上，在这个反应体系中，倘若水的量很大，在反应过程中没有发生变化，那么这时蔗糖的水解就是一个一级反应。

### 速率常数

以上的速率方程中有一个常数，我们称之为速率常数。其SI单位是。其物理意义是各反应物均为单位浓度时的反应速率，大小与反应物浓度无关。对于一个给定反应，的值由温度和催化剂决定。

### 反应分子数

基元反应中同时参加反应的“分子”数目称为该基元反应的反应分子数。

hexo插件简介

Hexo 有强大的插件系统，使您能轻松扩展功能而不用修改核心模块的源码。在 Hexo 中有两种形式的插件：

脚本（Scripts）

如果您的代码很简单，建议您编写脚本，您只需要把 JavaScript 文件放到 scripts 文件夹，在启动时就会自动载入。

插件（Packages）

如果您的代码较复杂，或是您想要发布到 NPM 上，建议您编写插件。首先，在 node_modules 文件夹中建立文件夹，文件夹名称开头必须为 hexo-，如此一来 Hexo 才会在启动时载入否则 Hexo 将会忽略它。
